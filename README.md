# Shopify Next.js Commerce Demo

This is a demo project showcasing a Shopify-powered e-commerce storefront built with Next.js and TypeScript.

## Features

- Shopify integration for product, cart, and checkout management
- Built with Next.js for fast, server-rendered React applications
- TypeScript for type safety and maintainability
- Modern e-commerce architecture—easily extensible

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Amankumar006/shopify-next-commerce-demo.git
   cd shopify-next-commerce-demo
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Configure environment variables:**
   - Copy `.env.example` to `.env.local` and fill in your Shopify credentials and other required values.

4. **Run the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

5. **Open your browser:**  
   Visit `http://localhost:3000` to see your store in action.

## Project Structure

- `pages/` — Next.js routes and pages
- `components/` — Reusable React components
- `lib/` — Shopify API interactions and utilities

## Requirements

- Node.js 18.x or above
- A Shopify store and API credentials
